**Step 5: Create `src/lib/gemini/cache.ts`** -- In-memory cache with 30-min TTL per D-03, D-04

Define response interface:
```typescript
export interface ExecutiveSummaryResult {
  lines: string[];
  generatedAt: string;
  cached: boolean;
}
```

Module-level cache variable:
```typescript
let cache: { data: ExecutiveSummaryResult; expiry: number } | null = null;
const TTL = 30 * 60 * 1000; // 30 minutes
```

`getOrRefreshSummary(): Promise<ExecutiveSummaryResult>`:
1. If `cache && Date.now() < cache.expiry` -> return `{ ...cache.data, cached: true }`
2. Else try:
   - Import `genai` from `./client` and `aggregateAllData` from `./data-aggregator` and `SYSTEM_INSTRUCTION`, `buildUserPrompt` from `./prompt`
   - If `!genai` throw `new Error("Gemini API not configured")`
   - Call `aggregateAllData()` to get data
   - Call `genai.models.generateContent({ model: "gemini-2.5-flash", contents: buildUserPrompt(data), config: { systemInstruction: SYSTEM_INSTRUCTION, temperature: 0.3, maxOutputTokens: 500 } })`
   - Parse: `const text = response.text ?? ""; const lines = text.split("\n").filter(l => l.trim().length > 0).slice(0, 3);`
   - Store in cache with `expiry: Date.now() + TTL`, return `{ lines, generatedAt: new Date().toISOString(), cached: false }`
3. On catch: if `cache` exists return `{ ...cache.data, cached: true }` (D-04: last successful fallback). Otherwise re-throw.

Export `getOrRefreshSummary` and `ExecutiveSummaryResult`.

**Step 6: Create `src/app/api/executive-summary/route.ts`** -- Following existing route pattern

```typescript
import { NextResponse } from "next/server";
import { getOrRefreshSummary } from "@/lib/gemini/cache";
import type { ExecutiveSummaryResult } from "@/lib/gemini/cache";

export async function GET(): Promise<NextResponse<ExecutiveSummaryResult | { error: string }>> {
  try {
    const result = await getOrRefreshSummary();
    return NextResponse.json(result);
  } catch (error) {
    console.error("[/api/executive-summary] failure:", error);
    return NextResponse.json(
      { error: "Failed to generate executive summary" },
      { status: 500 }
    );
  }
}
```

|-----------|----------|-----------|-------------|-----------------|
